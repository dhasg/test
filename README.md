# Asuogyaman District Health Directorate Website

Official website for the Asuogyaman District Health Directorate,
Asuogyaman District Assembly, Ghana.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- PHP (Contact Form Backend)

## Features
- Responsive design (mobile, tablet, desktop)
- Multilingual support (English, Twi, Ewe, Dangme)
- Accessible navigation
- Secure contact form with spam protection
- Backend email handling

## Deployment Instructions
1. Upload all files to the public_html directory.
2. Ensure PHP is enabled on the server.
3. Update email address in backend/contact.php.
4. Test contact form after deployment.
